# 🏦 Banking Management System

A console-based Banking Management System developed using **Python**, **SQL Server**, and **pyodbc**.

## 🚀 Features

- View Customers
- Add Customer
- Search Customer
- Delete Customer
- Create Account
- View Accounts
- Deposit Money
- Withdraw Money
- Check Balance
- Transfer Money
- View Transaction History

## 🛠️ Technologies Used

- Python
- SQL Server
- pyodbc
- Visual Studio Code
- Git & GitHub

## 📁 Project Structure

```
BankingSystem/
│── main.py
│── db.py
│── BankingSystem.sql
│── requirements.txt
│── README.md
```

## ⚙️ Installation

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run `BankingSystem.sql` in SQL Server.
4. Update the connection details in `db.py`.
5. Run:

```bash
python main.py
```
